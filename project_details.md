## Реализованные шаги

1. **Документация:**
   - Создан подробный файл [antigravity.md](file:///c:/Users/lev/Desktop/личное/хобби/devloper/1c%20auth/antigravity.md) с описанием архитектуры, целей, зависимостей и инструкций по работе с проектом.

2. **Согласованность версий:**
   - Файл `config.yaml` обновлен: `appVersion` установлен в `1.0.0`.
   - В `1c-auth.cs` обновлены атрибуты сборки `AssemblyVersion` и `AssemblyFileVersion` до версии `1.0.0`.
   - Логика свойства `Program.Version` изменена: удален фоллбэк на строку `"unknown"`. Теперь, если версия не задана или пуста в конфигурации, выбрасывается `Exception` с поясняющим сообщением.

3. **Оптимизация сборки:**
   - Удален файл `1c-auth.csproj`.
   - Написан новый скрипт [build.bat](file:///c:/Users/lev/Desktop/личное/хобби/devloper/1c%20auth/build.bat), использующий встроенный компилятор `csc.exe` (из `%WINDIR%\Microsoft.NET\Framework\v4.0.30319`). Он автоматически собирает исполняемый файл приложения вместе с файлом иконки.

4. **Выделение файла конфигурации:**
   - Класс конфигурации выделен в независимый файл [Config.cs](file:///c:/Users/lev/Desktop/личное/хобби/devloper/1c%20auth/Config.cs).
   - Класс переименован с `Config` в `AppConfig`.
   - Все ссылки в главном коде (`1c-auth.cs`) обновлены для использования `AppConfig`.

5. **Юнит-тесты:**
   - Вместо подключения тяжеловесных NuGet-пакетов xUnit/NUnit написан легковесный тестовый раннер, совместимый со стандартным компилятором `csc`. Это идеально подходит для работы на старых версиях Windows 10.
   - Тесты созданы в `1C_Authorizator_CS_TEST/Tests.cs`.
   - Написан скрипт запуска [build_tests.bat](file:///c:/Users/lev/Desktop/личное/хобби/devloper/1c%20auth/build_tests.bat), который автоматически компилирует и запускает тесты.
   - Тесты покрывают чтение настроек из файла (включая парсинг различных типов) и проверку генерации исключения при отсутствии версии. Все 4 базовых теста успешно пройдены.

## Инструкции по запуску

- **Для компиляции проекта:** выполните команду `.\build.bat` из командной строки.
- **Для запуска приложения:** после успешной компиляции запустите скомпилированный `1c-auth.exe`.
- **Для запуска тестов:** выполните `.\build_tests.bat`. Результаты будут выведены в консоль.
- **Для установки пользователю:** используйте `install.bat`. Он скопирует файлы в директорию профиля пользователя и добавит ярлык в автозагрузку.
