# 1C Authorizator

## Цели проекта
1C Authorizator — это приложение, предназначенное для быстрой и безопасной авторизации в системе 1С:Предприятие с помощью сканера штрих-кодов (или QR-кодов). Оно перехватывает ввод сканера на системном уровне, обрабатывает и валидирует данные авторизации, после чего прозрачно запускает клиент 1С с нужными параметрами.

## Архитектура
Приложение представляет собой легковесную утилиту на C# (Windows Forms, .NET Framework 4.8), работающую в фоновом режиме (системный трей).
Основные компоненты:
1. **Низкоуровневый перехватчик (Hook):** Использует Win32 API (`SetWindowsHookEx`) для отслеживания ввода с клавиатуры или сканера штрих-кодов, работающего в режиме эмуляции клавиатуры.
2. **Парсер и валидатор:** Буферизирует ввод, проверяет префикс сканера, очищает ввод от мусорных символов (с учетом различных раскладок клавиатуры) и декодирует Base32 -> JSON.
3. **Модуль запуска 1С:** Извлекает из JSON учетные данные и запускает `1cv8.exe` с аргументами командной строки, предварительно закрывая уже открытые экземпляры 1С, если это настроено.
4. **Интерфейс пользователя (UI):** Небольшое всплывающее окно для отображения статуса сканирования и ошибок, с поддержкой тем (светлая/темная) и интеграцией в системный трей.

## Зависимости
Проект спроектирован так, чтобы минимизировать внешние зависимости. Для компиляции и работы требуется только стандартный .NET Framework, встроенный в Windows.
- **.NET Framework 4.8** (встроен в Windows 10/11)
- **Windows Forms** (системная сборка `System.Windows.Forms.dll`)
- **System.Drawing** (системная сборка `System.Drawing.dll`)
Отсутствует необходимость в установке .NET SDK, NuGet-пакетов или сторонних библиотек.

## Инструкции по сборке
Для сборки проекта используется стандартный компилятор C# (`csc.exe`), поставляемый вместе с Windows. Сборка не требует установки Visual Studio или `dotnet` CLI.
1. Откройте командную строку или PowerShell.
2. Запустите скрипт `build.bat`, находящийся в корне репозитория.
3. Скрипт найдет компилятор в системной директории `%WINDIR%\Microsoft.NET\Framework\v4.0.30319\csc.exe` и скомпилирует `1c-auth.exe`.

## Тестирование
Для обеспечения совместимости со старыми версиями Windows без использования сторонних фреймворков (вроде xUnit/NUnit), проект использует легковесный встроенный механизм тестирования.
1. Запустите `build_tests.bat`.
2. Этот скрипт скомпилирует тестовое приложение и автоматически запустит его.
3. Результаты выполнения тестов (Passed/Failed) будут выведены в консоль. Проверяется загрузка конфигурации и обработка версионирования.

## Инструкции по публикации и установке
Установка на целевую машину максимально проста:
1. Выполните сборку через `build.bat`.
2. Запустите скрипт `install.bat`.
   - Он скопирует скомпилированный `1c-auth.exe`, `config.yaml` и иконки в папку `%LOCALAPPDATA%\1C_Authorizator`.
   - Создаст ярлык в папке "Автозагрузка" (`Startup`), чтобы приложение стартовало вместе с Windows.
3. Настройте параметры 1С и сканера в файле `config.yaml` в целевой директории.

## 4. Правила взаимодействия с AI (Antigravity)
1. **Документирование изменений:** Каждое изменение в коде должно сопровождаться комментарием `// [MODIFICATION]`, объясняющим *что* было изменено и *почему*.
2. **Язык общения:** Строго используй только русский язык при написании планов, отчетов, артефактов и общении в чате. Это поможет избежать повторения прошлых ошибок ("хождения по граблям") и сохранит историю принятых архитектурных решений.
